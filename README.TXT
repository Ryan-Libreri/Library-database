# 📚 Library Database System

A **Java-based Library Database System** developed as part of an intermediate-level computing assignment.  
It simulates core database operations for managing a library, including book records, user data, and borrowing functionalities.

---

## 🚀 Features

- 📖 Add, update, and delete book records  
- 👤 Manage user information  
- 🔄 Handle book borrowing and returns  
- 🖥️ Console-based user interface  
- 🧱 Built using object-oriented design principles  

---

## 🛠️ Built With

- **Java** – Main programming language
