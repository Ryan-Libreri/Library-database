# 📚 Library Database System

This project is a **Java-based Library Database System** developed as part of an intermediate-level computing assignment. It simulates basic database operations for managing a library system, including book records, users, and borrowing functions.

## 🚀 Features

- Add, update, and delete book records
- Manage user information
- Handle book borrowing and returns
- Console-based user interface
- Object-oriented design principles
